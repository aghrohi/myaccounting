Get Set Up

4-Step Installation

Get your accounting system running in minutes.

  1. Step 1: Database Setup

    Create a new MySQL database (e.g., `accounting_app`) and import the `database.sql` file.

    mysql -u your_username -p accounting_app < database.sql

  2. Step 2: Configure Connection

    Edit `db_connect.php` with your database credentials.

  3. Step 3: Upload Files

    Upload all application files to your web server directory (e.g., `/var/www/html/accounting/`).

  4. Step 4: Access Application

    Navigate to your server URL. You're ready to go!

Support & Future

Troubleshooting & Roadmap

Common Issues

  • Check database credentials in `db_connect.php`.
  • Ensure your MySQL service is running.
  • Verify the database `accounting_app` exists and the user has permissions.
  • Verify you are using the correct default credentials (`admin` / `Admin@123`).
  • Clear your browser cookies and session.
  • Ensure `mysqldump` is installed on your server and in the system's PATH.
  • Check that the PHP `passthru()` function is not disabled in `php.ini`.
